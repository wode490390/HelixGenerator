## Configurations

### config.yml
```yaml
border:
  # Block ID used to generate the border
  material: 155
  meta: 1
path:
  # Block ID used to generate the path
  material: 155
  meta: 3
```

## Compiling
1. Install [Maven](https://maven.apache.org/).
2. Run `mvn clean package`. The compiled JAR can be found in the `target/` directory.
